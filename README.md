##Pokédex Database App

A responsive Pokédex web application built with HTML, CSS, and JavaScript. The project uses the PokéAPI to dynamically fetch and display Pokémon data, including images, types, abilities, evolution chains, and detailed statistics.

Features
Integration with the PokéAPI
Dynamic data fetching using the JavaScript Fetch API
Pokémon search functionality
Responsive design for desktop and mobile devices
Loading screen while data is being retrieved
"Load More Pokémon" button to dynamically load additional Pokémon
Detailed Pokémon information page
Pokémon evolution chain display
Detailed Pokémon statistics with visual stat bars
Dynamic rendering of images, types, abilities, and stats
Interactive and user-friendly interface

##Technologies Used
HTML5
CSS3
JavaScript (ES6)
Fetch API
PokéAPI
What I Learned

This project helped me gain practical experience with:

Working with external REST APIs
Asynchronous JavaScript (Promises & Fetch API)
Handling and processing large amounts of API data
Dynamic DOM manipulation
Loading states and user feedback
Rendering complex data structures
Building responsive user interfaces
Visualizing data through dynamic stat bars
